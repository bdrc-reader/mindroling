# སྔ་འགྱུར་མཐོ་སློབ་མདོ་རྒྱུད་ལེགས་བཤད་གླིང་། 

<img src="https://user-images.githubusercontent.com/42564256/89265246-c6e31300-d666-11ea-97ee-1a166d365505.jpg" align="right" width="350" height="250">


<br>

　　མཚན་རྒྱས་པར་“འོག་མིན་ཨོ་རྒྱན་སྨིན་གྲོལ་གླིང་”ཞེས་ཞུ། གྲ་ཐང་དགོན་ནས་ཤར་དུ་སྣུམ་ནག་འཁོར་ལམ་དེད་སྤྱི་ལེ་བཞི་ལྔ་ཙམ་སོང་ན་ཚོང་འདུས་གྲོང་དུ་འབྱོར། དེ་ནས་ནུབ་ངོས་རི་འགྲམ་ནས་འཁོར་ལམ་སྤྱིར་བཏང་བ་བརྒྱུད་ལྷོ་རུ་སྤྱི་ལེ་བརྒྱད་དགུ་ཙམ་སོང་ན་ཐར་པ་གླིང་གྲོང་དབུས་སུ་དགོན་དེ་ཆགས་ཡོད། དགོན་དེ་ནི་རྡོ་རྗེ་བྲག་དང་། སྨིན་གྲོལ་གླིང་། འཕྱོངས་རྒྱས་དཔལ་རི་བཅས་ལྷོ་ཁུལ་གྱི་རྙིང་དགོན་ཆེ་བ་གསུམ་གྱི་ཡ་གྱལ་ཡིན། ཐོག་མར་གནས་དེར་ཀླུ་མེད་ཚུལ་ཁྲིམས་ཤེས་རབ་ཀྱི་དགོན་པ་“ཐར་པ་གླིང་”ཞེས་པ་ཞིག་ཡོད་སྟབས་དེ་ཁུལ་གྱི་གྲོང་ལའང་ཐར་པ་གླིང་ཞེས་པའི་མིང་ཐོགས། དེ་རྗེས་རྒྱལ་དབང་ལྔ་པ་ཆེན་པོས་ཐར་པ་གླིང་གི་བདག་དབང་གཏེར་བདག་གླིང་པར་གནང་བས་༡༦༧༦ལོར་སྨིན་གླིང་དགོན་བཞེངས་འགོ་ཚུགས། གཙུག་ལག་ཁང་ཀ་བ་བརྒྱད་ཅུ་གྱ་བཞིའི་ཐོག་བརྩེགས་གསུམ་པའི་ཡང་སྟེང་གསར་ཟངས་ཀྱི་གཉྫི་ར་དང་། ལི་མ་ལྷ་ཁང་གི་ཡངས་སྟེང་གསེར་ཟངས་ཀྱི་རྒྱལ་མཚན་གཉིས། ཕྱི་རོལ་དུ་འཁོར་ས་བྲང་ཁང་དང་བཅས་ལེགས་པར་གྲུབ། གོང་ས་མཆོག་ནས་དངུལ་དཀར་གྱི་བདེ་གཤེགས་མཆོད་རྟེན་བརྒྱད་ཐོག་[page 33] རྩེགས་མའང་གསོལ་སྩལ་ཐོབ་པ་སོགས་རྟེན་གསུམ་ཁྱད་པར་ཅན་བཞུགས་སོ་གསོལ། ཐོག་མར་གྲྭ་གྲངས་བཞི་བཅུ་སྐོར་ནས་རིམ་བཞིན་སུམ་བརྒྱར་ཉེ་བ་བྱུང་། ༡༧༡༤ལོར་གཏེར་བདག་གླིང་པ་སྐུ་གཤེགས། ༡༧༡༧ལོར་ཇུན་གར་དམག་གིས་གསང་སྔགས་ཕོ་བྲང་མ་གཏོགས་ཡོངས་རྫོགས་བཤིགས། དེ་རྗེས་རྒྱལ་བ་སྐལ་བཟང་རྒྱ་མཚོའི་དུས་སླར་ཡང་རྙིང་དགོན་ཐོར་ཞིག་སོང་རིགས་ཉམས་གསོ་དགོས་པའི་བཀའ་ཤོག་ལྟར་རིམ་བཞིན་སྔར་ལྟར་ཉམས་གསོ་གནང་། གཏེར་བདག་གླིང་པའི་སྐུ་མཆེད་ལོ་ཆེན་དྷརྨ་ཤྲཱིས་རིག་གནས་སྤྱི་ལ་མཛད་པ་རྒྱ་ཆེན་སྤེལ་གནང་མཛད་པ་སྤྱི། བྱེ་བྲག་སྒྲ། གསོ་རིག སྐར་རྩིས་སོགས་ལ་མཁས་པར་གྱུར་པའི་རྒྱུན་ལས་སྔོན་དགོན་དེར་རིག་གནས་ཀྱི་སློབ་གྲྭ་ཞིག་ཡོད་པ་ལ་ཕྱོགས་མཐའ་ཁག་ནས་སེར་སྐྱ་རིས་མེད་སློབ་གཉེར་དུ་ཕེབས་སྲོལ་ཡོད།

　　ད་ལྟ་ཉམས་གསོ་གནང་བའི་འདུ་ཁང་དང་གཙུག་ལག་ཁང་ནང་རྟེན་གཙོར་ཐུབ་དབང་མཐོང་གྲོལ་ཐོག་རྩེགས་གསུམ་ལྡན་ཉེ་སྲས་བརྒྱད་ཀྱིས་བསྐོར་བ་དང་། གཏེར་བདག་གླིང་པས་གཏེར་ནས་བཞེས་པའི་ནང་རྟེན་ཁག་གཞན་ཡང་གསར་བཞེངས་སྐུ་འདྲ་མང་པོ་སོགས་མཇལ་རྒྱུ་ཡོད།

ལོ་ཆེན་དྷརྨ་ཤྲཱི་མཆོག་ནི་རབ་བྱུང་བཅུ་གཅིག་པའི་ཤིང་རྟ་ཕྱི་ལོ་༡༦༥༤ ལོར་གཡོར་པོ་གྲྭ་ནང་གི་དར་རྒྱས་ཆོས་གླིང་དུ་ཡབ་སྨྱོས་སྟོན་འཕྲིན་ལས་ལྷུན་གྲུབ་དང་། ཡུམ་ལྷ་འཛིན་དབྱངས་ཅན་སྒྲོལ་མ་གཉིས་ཀྱི་སྲས་སུ་འཁྲུངས། བྱིས་དུས་ཀྱི་མཚན་ལ་ལེགས་པ་དོན་གྲུབ་ཅེས་ཞུ། དགུང་ལོ་བརྒྱད་པར་གཅེན་པོ་དང་མཉམ་དུ་འཕྱོངས་རྒྱས་དཔལ་རི་དང་གནས་ཆེན་མཆིམས་ཕུར་བཞུགས། བཅུ་གསུམ་པར་གཏེར་སྟོན་བླ་མར་གཙུག་ཕུད་ཕུལ། མཚན་ལ་བསྟན་འཛིན་འཇམ་དབྱངས་དབང་པོ་ཞེས་སྩལ། བཅོ་ལྔ་པར་ལྷ་སར་རྒྱལ་དབང་ལྔ་པ་ལས་རབ་ཏུ་བྱུང་། མཚན་ལ་ངག་དབང་ཆོས་དཔལ་ཞེས་བཏགས། བཅུ་དགུ་པར་སློབ་དཔོན་དོན་གྲུབ་དབང་རྒྱལ་ལས་རྩིས་དང་། སྟག་ལུང་གྲུབ་ཆེན་ངག་དབང་གཟུགས་མཆོག་ལས་སྒྲའི་བསྟན་བཅོས་དང་། གཏེར་ཆེན་བླ་མ་ལས་སྙན་ངག་དང་སྡེབ་སྦྱོར་སོགས་གསན། ཉི་ཤུ་པར་རྒྱལ་དབང་ལྔ་པ་ལས་བསྙེན་པར་རྫོགས། སྨིན་གྲོལ་གླིང་གྲྭ་ཚང་གི་དགེ་སྐོས་ཀྱི་འགན་ཁུར། སྒྲིག་ལམ་དང་ཀུན་སྤྱོད་བཅས་ལ་དོ་དམ་མཛད། ཉེར་བདུན་པར་རྒྱལ་དབང་ལྔ་པའི་བཀའ་ཕེབས་པ་ལྟར་མཁན་ཆེན་དཀོན་མཆོག་བསྟན་འཛིན་ལས་འདུལ་བ་སྨད་ལུགས་ཀྱི་ཆོ་གས་བསྙེན་པར་རྫོགས། ཕྱི་ནང་ཐུན་མོང་དང་ཐུན་མོང་མ་ཡིན་པའི་རིག་གནས་ཀུན་ལ་མཁས་པར་སྦྱངས། རབ་བྱུང་བཅུ་གཉིས་པའི་ས་ཁྱི་ཕྱི་ལོ་༡༧༡༨ ལོར་ཨོ་རོད་ཇུང་གར་བས་སྐྱིད་ཆུའི་འགྲམ་དུ་བཀྲོངས། སློབ་མ་གཙོ་བོ་ནི་ས་སྐྱ་བདག་ཆེན་ངག་དབང་ཀུན་དགའ་བཀྲ་ཤིས་དང་། ཡར་འབྲོག་གླིང་གི་ཞབས་དྲུང་ཡབ་སྲས། ཕུན་ཚོགས་གླིང་གི་དགེ་སློང་འབྱུང་གནས། སྟག་སྣ་ཏྲེ་བོ་སྤྲུལ་པའི་སྐུ། ཆབ་མདོ་འཕགས་ལྷ་རྒྱལ་བ་རྒྱ་མཚོ། སྡེ་དགེའི་བླ་མ་རབ་འབྱམས་པ། མི་དབང་ཕོ་ལྷ་བ་བསོད་ནམས་སྟོབས་རྒྱལ། མདོ་མཁར་ཞབས་དྲུང་ཚེ་རིང་དབང་རྒྱལ་བཅས་བྱོན། གསུང་རྩོམ་ནི་གསང་བདག་དགོངས་རྒྱན་དང་། གསང་བདག་ཞལ་ལུང་། དཀྱིལ་འཁོར་བློས་བསླངས་ཀྱི་རྣམ་གཞག་གསལ་སྒྲོན་རྩ་འགྲེལ། སྡེབ་སྦྱོར་གྱི་འགྲེལ་བ་དོན་གསལ་མེ་ལོང་། སྐར་རྩིས་ཉིན་བྱེད་སྣང་བ་རྩ་འགྲེལ། དབྱངས་འཆར་གྱི་མཆན་འགྲེལ་རྒྱུ་སྐར་ཕྲེང་བ། འབྱུང་རྩིས་ཟླ་བའི་འོད་ཟེར། དབང་ཆོག་རྡོ་རྗེའི་ཐེམ་སྐས། སྡོམ་གསུམ་དཔག་བསམ་སྙེ་མ། འདུལ་བའི་ལས་ཆོག་རིན་ཆེན་ཐེམ་སྐས། གཏེར་བདག་གླིང་པའི་རྣམ་ཐར། འཆད་ཉན་བྱེད་ཚུལ་གྱི་ཡི་གེ་མདོར་བསྡུས། འབེལ་གཏམ། རྨི་ལམ་གྱི་ཁྲིད་ཡིག་ཉིང་འཁྲུལ་རང་གྲོལ། གླུ་ཡིག་རབ་གསལ་སྣང་བ། མངོན་བརྗོད་རྒྱ་མཚོའི་ཆུ་ཐིགས། སྙན་ངག་རྒྱ་མཚོའི་སྙིང་པོ་བཅས་ཆོས་ཚན་བདུན་ཅུ་ལྷག་བཞུགས། མིང་གི་རྣམ་གྲངས། ལེགས་པ་དོན་གྲུབ། བསྟན་འཛིན་འཇམ་དབྱངས་དབང་པོ། ངག་དབང་ཆོས་དཔལ། སྨིན་གླིང་ལོ་ཆེན།

སྨིན་གླིང་གཏེར་ཆེན་པདྨ་བཀའ་དབང་འགྱུར་མེད་རྡོ་རྗེ་མཆོག་ནི་རབ་བྱུང་བཅུ་གཅིག་པའི་མེ་ཕོ་ཁྱི་ལོ་སྟེ་སྤྱི་ལོ1646ལོར་གྲྭ་ནང་དར་རྒྱས་ཆོས་གླིང་དུ་ཡབ་སྨྱོས་སྟོན་གསང་བདག་འཕྲིན་ལས་ལྷུན་གྲུབ་དང་། ཡུམ་ལྷ་འཛིན་དབྱངས་ཅན་སྒྲོལ་མ་གཉིས་ཀྱི་སྲས་སུ་སྐུ་འཁྲུངས། རབ་བྱུང་བཅུ་གཉིས་པའི་ཤིང་ཕོ་རྟ་ལོ་སྟེ་སྤྱི་ལོ1714ལོར་སྐུ་གཤེགས།དགུང་ལོ་བཞི་པར་ཡབ་ཆེན་ལས་བཀའ་བརྒྱད་གསང་རྫོགས་ཀྱི་དབང་ལུང་རྣམས་གསན། དགུང་ལོ་བཅུ་གཅིག་པར་ལྷ་སར་ཕེབས་ཏེ་རྒྱལ་དབང་ལྔ་པ་ཆེན་པོར་དབུ་སྐྲའི་གཙུག་ཕུད་ཕུལ། དགུང་ལོ་བཅུ་གསུམ་པར་ཡོངས་སུ་གྲགས་པའི་སློབ་དཔོན་སུམ་ཅུ་རྩ་ལྔར་བསྟེན་ནས་མདོ་སྔགས་རིག་གནས་དང་བཅས་པར་ལེགས་པར་སྦྱངས་ནས་མཁས་པའི་གྲགས་པ་ཐོབ། དགུང་ལོ་བཅོ་བརྒྱད་པར་གཡའ་མ་ལུང་ནས་རིག་འཛིན་ཐུག་ཐིག་གི་ཆོས་སྐོར་དང་། དགུང་ལོ་ཉེར་གཉིས་པར་རྫུ་འཕྲུལ་གྱི་བཀོད་པས་ཤེལ་བྲག་ནས་འཇམ་དཔལ་གཤིན་རྗེ་དྲེགས་འཇོམས་དང་། དགུང་ལོ་སོ་གཅིག་པར་ཨོ་དཀར་ཕྲགས་ནས་གུར་དྲགས་དང་། རྡོར་སེམས།ཨ་ཏིའི་སྐོར། དགུང་ལོ་སོ་ལྔ་པར་ཤ་འུག་ལྟག་སྒོ་ནས་ཐུགས་རྗེ་ཆེན་པོ་བདེ་གཤེགས་ཀུན་འདུས་ཀྱི་ཆོས་སྐོར་སྤྱན་དྲངས་པ་སོགས་ཤིན་ཏུ་མང་སྐད། དགུང་ལོ་སོ་གཅིག་པ་རབ་བྱུང་བཅུ་གསུམ་པའི་མེ་ཕོ་འབྲུག་ལོ་སྟེ་སྤྱི་ལོ1676ལོར་སྨིན་གྲོལ་གླིང་གསར་འདེབས་མཛད་དེ་རྟེན་དང་རྟེན་པ་བཞེངས། དགུང་ལོ་ང་ལྔ་པར་སྒྲུབ་མཆོད་ཁང་དང་། གསང་སྔགས་ཕོ་བྲང་བཅས་གསར་བཞེངས་མཛད། བཤད་སྒྲུབ་ལས་གསུམ་གྱི་སྒོ་ནས་བསྟན་པ་རྒྱ་ཆེར་སྤེལད་ཏོ། ཁོང་གི་མཚན་གྱི་རྣམ་གྲངས་ནི་ཀུན་དགའ་དངོས་གྲུབ་རིན་ཆེན་དབང་རྒྱལ་དང་། པདྨ་གར་དབང་འགྱུར་མེད་རྡོ་རྗེ། ངག་དབང་པདྨ་བསྟན་འཛིན་བཅས་སོ། ཁོང་གི་སློབ་མ་གྲགས་ཆེ་བ་ནི་རྒྱལ་དབང་ལྔ་པ་ཆེན་པོ། སྡེ་སྲིད་སངས་རྒྱས་རྒྱ་མཚོ།རྡོ་བྲག་རིག་འཛིན་པདྨ་འཕྲིན་ལས། ས་སྐྱ་ཁྲི་ཆེན་ཀུན་དགའ་བཀྲ་ཤིས། རྩེ་གདོང་ཞབས་དྲུང་ཁུ་དབོན་བཅས་ཡིན། ཁོང་གི་གསུང་རྩོམ་ནི་པོད་ཆེན་བཅོ་ལྔ་ཙམ་བཞུགས་སོ།།​​​​
