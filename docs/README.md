# སྔ་འགྱུར་མཐོ་སློབ་མདོ་རྒྱུད་ལེགས་བཤད་གླིང་། 

<img src="https://user-images.githubusercontent.com/42564256/92857139-d7856680-f426-11ea-9c89-b2f482fcd48e.png" align="right" width="400" height="250">

　　དེ་ཡང་ཕྱི་ལོ་༡༩༨༨ ལོའི་སྔ་རོལ་དུ་སྔ་འགྱུར་མཐོ་སློབ་མདོ་རྒྱུད་ལེགས་བཤད་གླིང་འདི་ཉིད་ཕྱག་བཏབ་མེད་ནའང་རང་དགོན་དུ་སྔ་འགྱུར་རིག་གཞུང་སློབ་གཉེར་ཁང་ཞེས་པའི་བཤད་གྲྭ་ཆུང་ངུ་ཞིག་སྔོན་ནས་ཆགས་ཡོད། དེར་བདག་ཅག་རྣམས་ལ་བཀའ་དྲིན་བླ་ན་མ་མཆིས་པ་རྫོགས་ཆེན་མཁན་པོ་ཐུབ་བསྟན་བརྩོན་འགྲུས་ཕུན་ཚོགས་མཆོག་ནས་ལོ་ངོ་ལྔ་ལྷག་ཙམ་བཞུགས་རིང་སློབ་ཕྲུག་རྣམས་ལ། ཚད་མ། སྤྱོད་འཇུག །ཤེར་ཕྱིན། དབུ་མ། མཁས་འཇུག་སོགས་ནང་དོན་རིག་པའི་གཞུང་ལུགས་ཉིན་ལྟར་དཔེ་ཁྲིད་ཆག་མེད་གནང་། སྐབས་དེ་དུས་ཀྱི་སློབ་མའི་གཙོ་བོ་པད་བཀོད་རྒྱལ་སྲས་སྤྲུལ་སྐུ་ཐུབ་བསྟན་ཤེས་རབ་གྲགས་པ་དང་། མཁན་ཆེན་པདྨ་ཤེས་རབ། མཁན་པོ་འགྱུར་མེད་ཚུལ་ཁྲིམས་སོགས་ཡིན། དེ་རྗེས་རྒྱས་སྲས་སྤྲུལ་སྐུ་མཆོག་ནས་ལོ་ངོ་བཅུའི་རིང་དཔེ་ཁྲིད་ཀྱི་བཀའ་དྲིན་བསྐྱངས། སླར་ཡང་རིགས་འདུས་པདྨའི་རྒྱལ་ཚབ་བདེ་བར་གཤེགས་པ་༧སྐྱབས་རྗེ་སྨིན་གླིང་ཁྲི་ཆེན་རྡོ་རྗེ་འཆང་ཆེན་པོ་མཆོག་དང་། དམ་པའི་ཚུལ་དགུས་བརྒྱན་པ་ྋསྐྱབས་རྗེ་མཁོ་ཁྱིམ་རིན་པོ་ཆེ་འགྱུར་མེད་མདོ་རྒྱུད་རྒྱ་མཚོ་མཆོག་གི་ཐུགས་བསྐྱེད་བླ་ན་མེད་པ་ལ་བརྟེན་ནས། སྤྱིར་འཕགས་ཡུལ་གྱི་སློབ་དཔོན་ཆེན་པོ་རྒྱན་དྲུག་མཆོག་གཉིས་དང་ཁྱད་པར་དུ་གསང་ཆེན་སྔ་འགྱུར་རྙིང་མའི་ཤིང་རྟ་ཆེན་པོ་ཀུན་མཁྱེན་རོང་ཀློང་རྣམ་གཉིས། སྨིན་གླིང་ལོ་ཆེན་དྷརྨ་ཤྲཱི། སྨིན་གླིང་ཁྲི་རབས་གཉིས་པ་པདྨ་འགྱུར་མེད་རྒྱ་མཚོ། ཀུན་མཁྱེན་མི་ཕམ་ཕྱོགས་ལས་རྣམ་རྒྱལ། རྒྱལ་སྲས་གཞན་ཕན་ཆེན་ཆོས་ཀྱི་སྣང་བ་སོགས་ཀྱི་དགོངས་པ་ལྟར་རང་ལུགས་ཀྱི་ལྟ་གྲུབ་ལ་རང་རྐང་ཚུགས་པའི་བཤད་ཁྲིད་དང་། གསང་ཆེན་སྔ་འགྱུར་རྙིང་མའི་མདོ་སྔགས་ཀྱི་ཟབ་ཆོས་རྣམས་མི་ཉམས་པར་བཤད་སྒྲུབ་ཀྱི་ཚུལ་ལ་དམིགས་ཏེ་ཕྱི་ལོ་༡༩༨༨ ལོའི་ཟླ་བ་༤ཚེས་༢༦ཉིན་གྱི་དུས་ཚེས་དགེ་བར་སྔ་འགྱུར་མཐོ་སློབ་མདོ་རྒྱུད་ལེགས་བཤད་གླིང་གསར་དུ་ཕྱག་བཏབ། དེའི་སྐབས་༧གོང་ས་ྋསྐྱབས་མགོན་ཆེན་པོ་མཆོག་ཞབས་སོར་འཁོད་ནས་བཤད་གྲྭའི་རྨང་རྡོ་བཙུགས་ཤིང་ཐུགས་སྨོན་རབ་གནས་ཟབ་རྒྱས་མཛད། 
དེ་རྗེས་དབུ་འབྱེད་རྟེན་འབྲེལ་ཐོག་སྔ་འགྱུར་བསྟན་པའི་གཙུག་རྒྱན་༧སྐྱབས་རྗེ་སྨིན་གླིང་ཁྲི་ཆེན་རིན་པོ་ཆེ་མཆོག་དང་༧སྐྱབས་རྗེ་གྲུབ་དབང་པད་ནོར་རིན་པོ་ཆེ། སྨིན་གླིང་མཁན་ཆེན་རིན་པོ་ཆེ།ཞེ་ཆེན་རབ་འབྱམས་རིན་པོ་ཆེ། མཁོ་ཁྱིམ་རིན་པོ་ཆེ་དང་། གཞན་ཡང་ཆོས་ལུགས་ཁག་གི་སྐུ་ཚབ་གཙོས་ཕྱོགས་ཀྱི་སྐུ་མགྲོན་བླ་སྤྲུལ་མཁན་པོ་དགེ་འདུན་པ་རྣམས་ཀྱིས་རབ་གནས་དང་བཀྲ་ཤིས་པའི་དབུ་འབྱེད་ཀྱི་རྟེན་འབྲེལ་ལེགས་པར་མཛད། གསར་ཞུགས་སློབ་ཕྲུག་གྲངས་ཉེར་ལྔ་ལ་ཐོག་མའི་དཔེ་ཁྲིད་ནི། སྨིན་གླིང་མཁན་སྤྲུལ་རིན་པོ་ཆེ་མཆོག་དང་། པད་བཀོད་རྒྱལ་སྲས་རིན་པོ་ཆེ།  མཁན་པོ་བློ་གྲོས་དྲི་མེད། མཁན་པོ་བསྟན་འཛིན་བསམ་འཕེལ་བཅས་ཀྱིས་གནང་། དེ་ནས་རིམ་བཞིན་མཁན་ཆེན་བདེ་བར་གཤེགས་པ་ཐུབ་བསྟན་འོད་ཟེར་མཆོག་དང་།མཁན་རིན་པོ་ཆེ་རྣམ་གྲོལ་ཚེ་རིང་། མཁན་པོ་དཀོན་མཆོག་འོད་ཟེར། མཁན་པོ་འཇམ་དཔལ་ཚེ་རིང་། མཁན་པོ་རྟ་མགྲིན་སྲིད་ཐུབ།མཁན་པོ་ཚེ་དབང་རིག་འཛིན། མཁན་པོ་དགེ་ལེགས་བསྟན་འཛིན།མཁན་པོ་སྐལ་བཟང་བསོད་ནམས་བཅས་དང་།ལྷག་པར་དུ་ཕྱི་ལོ་༡༩༩༣ ལོར་སྤྱི་ནོར་ྋགོང་ས་ྋསྐྱབས་མགོན་ཆེན་པོ་མཆོག་ཞབས་སོར་འཁོད་ཅིང་སྒྱུ་འཕྲུལ་གསང་སྙིང་གི་འགྲེལ་པ་ཕྱོགས་བཅུའི་མུན་སེལ་གྱི་ལྗགས་ཁྲིད་རྟེན་འབྲེལ་དབུ་འབྱེད་གནང་། སྔ་འགྱུར་བསྟན་པའི་བདག་པོ་༧སྐྱབས་རྗེ་སྨིན་གླིང་ཁྲི་ཆེན་རིན་པོ་ཆེ་མཆོག་ནས་ཡོན་ཏན་མཛོད་ཀྱི་བཤད་ཁྲིད། ༧སྐྱབས་རྗེ་མཁོ་ཁྱིམ་རིན་པོ་ཆེ་ནས་སྡོམ་གསུམ་དཔག་བསམ་སྙེ་མ། གསང་བདག་ཞལ་ལུང་དང་དགོངས་རྒྱན། ཤེར་ཕྱིན་བདུད་རྩི་ཉིང་ཁུ་སོགས་ཀྱི་བཤད་ལུང་བསྩལ། དེ་ནས་རིམ་བཞིན་སྔ་འགྱུར་མཐོ་སློབ་ཁང་འདི་ནསབསླབ་པ་མཐར་སོན་པའི་མཁན་སློབ་རྣམས་ཀྱིས་རང་དགོན་དང་རྒྱ་བལ་འབྲུག་འབྲས་སོགས་ཕྱོགས་མཐའ་ཁག་ལ་དཔེ་ཁྲིད་ཀྱི་ཞབས་ཞུ་སོགས་རབ་དང་རིམ་པས་ད་ལྟའི་ཆར་གནང་དང་གནང་མུས་ཡིན། སླར་ཡང་ཕྱི་ལོ་༢༠༡༢ ལོ་ནས་སྨིན་གླིང་མཁན་ཆེན་རིན་པོ་ཆེའི་ཐུགས་བསྐྱེད་བླ་ན་མེད་པ་ལ་བརྟེན་ནས་དེང་དུས་སློབ་གཉེར་བའི་གྲངས་འབོར་སྔར་བས་མང་བའི་རྐྱེན་གྱིས། སྔོན་གྱི་བཤད་གྲྭའི་གཙུག་ལག་ཁང་དང་གྲྭ་ཤག་སོགས་གཞི་རྒྱ་ཆུང་དྲགས་པའི་དཀའ་ངལ་ཡོད་པར་བརྟེན་ཧ་ཅང་གཞི་རྒྱ་ཆེ་བའི་བཤད་གྲྭའི་བཞུགས་ཤག་དང་། ཚོགས་ཁང་། དཔེ་མཛོད་ཁང་བཅས་གསར་རྒྱག་དབུ་བཙུགས་ནས་ད་ལྟའི་ཆར་ལ་བཞུགས་ཤག་དང་མདོ་སྔགས་རིག་པའི་གཙུག་ལག་ཁང་དང་དཔེ་མཛོད་ཟུང་འབྲེལ་བཅས་ལེགས་པར་གྲུབ་ཟིན།མདོར་ན་བསྟན་འཛིན་གྱི་སྐྱེས་ཆེན་དུ་མས་མདོ་སྔགས་རིག་གནས་དང་བཅས་པའི་གཞུང་ལུགས་ལ་ཐོས་བསམ་སྒོམ་གསུམ་གྱི་སྒོ་ནས་རྒྱལ་བའི་བསྟན་པ་ལ་བྱ་བ་མི་དམན་པའི་ཞབས་འདེགས་ཞུ་བཞིན་མཆིས་པ་བཅས་སོ།།


## ལོ་ཆེན་དྷརྨ་ཤྲཱི།

<img src="https://user-images.githubusercontent.com/42564256/89266204-31e11980-d668-11ea-999e-1f7b7aef0af5.jpg" align="right" width="150" height="200">

ལོ་ཆེན་དྷརྨ་ཤྲཱི་མཆོག་ནི་རབ་བྱུང་བཅུ་གཅིག་པའི་ཤིང་རྟ་ཕྱི་ལོ་༡༦༥༤ ལོར་གཡོར་པོ་གྲྭ་ནང་གི་དར་རྒྱས་ཆོས་གླིང་དུ་ཡབ་སྨྱོས་སྟོན་འཕྲིན་ལས་ལྷུན་གྲུབ་དང་། ཡུམ་ལྷ་འཛིན་དབྱངས་ཅན་སྒྲོལ་མ་གཉིས་ཀྱི་སྲས་སུ་འཁྲུངས། བྱིས་དུས་ཀྱི་མཚན་ལ་ལེགས་པ་དོན་གྲུབ་ཅེས་ཞུ། དགུང་ལོ་བརྒྱད་པར་གཅེན་པོ་དང་མཉམ་དུ་འཕྱོངས་རྒྱས་དཔལ་རི་དང་གནས་ཆེན་མཆིམས་ཕུར་བཞུགས། བཅུ་གསུམ་པར་གཏེར་སྟོན་བླ་མར་གཙུག་ཕུད་ཕུལ། མཚན་ལ་བསྟན་འཛིན་འཇམ་དབྱངས་དབང་པོ་ཞེས་སྩལ། བཅོ་ལྔ་པར་ལྷ་སར་རྒྱལ་དབང་ལྔ་པ་ལས་རབ་ཏུ་བྱུང་། མཚན་ལ་ངག་དབང་ཆོས་དཔལ་ཞེས་བཏགས། བཅུ་དགུ་པར་སློབ་དཔོན་དོན་གྲུབ་དབང་རྒྱལ་ལས་རྩིས་དང་། སྟག་ལུང་གྲུབ་ཆེན་ངག་དབང་གཟུགས་མཆོག་ལས་སྒྲའི་བསྟན་བཅོས་དང་། གཏེར་ཆེན་བླ་མ་ལས་སྙན་ངག་དང་སྡེབ་སྦྱོར་སོགས་གསན། ཉི་ཤུ་པར་རྒྱལ་དབང་ལྔ་པ་ལས་བསྙེན་པར་རྫོགས། སྨིན་གྲོལ་གླིང་གྲྭ་ཚང་གི་དགེ་སྐོས་ཀྱི་འགན་ཁུར། སྒྲིག་ལམ་དང་ཀུན་སྤྱོད་བཅས་ལ་དོ་དམ་མཛད། ཉེར་བདུན་པར་རྒྱལ་དབང་ལྔ་པའི་བཀའ་ཕེབས་པ་ལྟར་མཁན་ཆེན་དཀོན་མཆོག་བསྟན་འཛིན་ལས་འདུལ་བ་སྨད་ལུགས་ཀྱི་ཆོ་གས་བསྙེན་པར་རྫོགས། ཕྱི་ནང་ཐུན་མོང་དང་ཐུན་མོང་མ་ཡིན་པའི་རིག་གནས་ཀུན་ལ་མཁས་པར་སྦྱངས། རབ་བྱུང་བཅུ་གཉིས་པའི་ས་ཁྱི་ཕྱི་ལོ་༡༧༡༨ ལོར་ཨོ་རོད་ཇུང་གར་བས་སྐྱིད་ཆུའི་འགྲམ་དུ་བཀྲོངས། སློབ་མ་གཙོ་བོ་ནི་ས་སྐྱ་བདག་ཆེན་ངག་དབང་ཀུན་དགའ་བཀྲ་ཤིས་དང་། ཡར་འབྲོག་གླིང་གི་ཞབས་དྲུང་ཡབ་སྲས། ཕུན་ཚོགས་གླིང་གི་དགེ་སློང་འབྱུང་གནས། སྟག་སྣ་ཏྲེ་བོ་སྤྲུལ་པའི་སྐུ། ཆབ་མདོ་འཕགས་ལྷ་རྒྱལ་བ་རྒྱ་མཚོ། སྡེ་དགེའི་བླ་མ་རབ་འབྱམས་པ། མི་དབང་ཕོ་ལྷ་བ་བསོད་ནམས་སྟོབས་རྒྱལ། མདོ་མཁར་ཞབས་དྲུང་ཚེ་རིང་དབང་རྒྱལ་བཅས་བྱོན། གསུང་རྩོམ་ནི་གསང་བདག་དགོངས་རྒྱན་དང་། གསང་བདག་ཞལ་ལུང་། དཀྱིལ་འཁོར་བློས་བསླངས་ཀྱི་རྣམ་གཞག་གསལ་སྒྲོན་རྩ་འགྲེལ། སྡེབ་སྦྱོར་གྱི་འགྲེལ་བ་དོན་གསལ་མེ་ལོང་། སྐར་རྩིས་ཉིན་བྱེད་སྣང་བ་རྩ་འགྲེལ། དབྱངས་འཆར་གྱི་མཆན་འགྲེལ་རྒྱུ་སྐར་ཕྲེང་བ། འབྱུང་རྩིས་ཟླ་བའི་འོད་ཟེར། དབང་ཆོག་རྡོ་རྗེའི་ཐེམ་སྐས། སྡོམ་གསུམ་དཔག་བསམ་སྙེ་མ། འདུལ་བའི་ལས་ཆོག་རིན་ཆེན་ཐེམ་སྐས། གཏེར་བདག་གླིང་པའི་རྣམ་ཐར། འཆད་ཉན་བྱེད་ཚུལ་གྱི་ཡི་གེ་མདོར་བསྡུས། འབེལ་གཏམ། རྨི་ལམ་གྱི་ཁྲིད་ཡིག་ཉིང་འཁྲུལ་རང་གྲོལ། གླུ་ཡིག་རབ་གསལ་སྣང་བ། མངོན་བརྗོད་རྒྱ་མཚོའི་ཆུ་ཐིགས། སྙན་ངག་རྒྱ་མཚོའི་སྙིང་པོ་བཅས་ཆོས་ཚན་བདུན་ཅུ་ལྷག་བཞུགས། མིང་གི་རྣམ་གྲངས། ལེགས་པ་དོན་གྲུབ། བསྟན་འཛིན་འཇམ་དབྱངས་དབང་པོ། ངག་དབང་ཆོས་དཔལ། སྨིན་གླིང་ལོ་ཆེན།

སྨིན་གླིང་གཏེར་ཆེན་པདྨ་གར་དབང་འགྱུར་མེད་རྡོ་རྗེ་མཆོག་ནི་རབ་བྱུང་བཅུ་གཅིག་པའི་མེ་ཕོ་ཁྱི་ལོ་སྟེ་སྤྱི་ལོ1646ལོར་གྲྭ་ནང་དར་རྒྱས་ཆོས་གླིང་དུ་ཡབ་སྨྱོས་སྟོན་གསང་བདག་འཕྲིན་ལས་ལྷུན་གྲུབ་དང་། ཡུམ་ལྷ་འཛིན་དབྱངས་ཅན་སྒྲོལ་མ་གཉིས་ཀྱི་སྲས་སུ་སྐུ་འཁྲུངས། རབ་བྱུང་བཅུ་གཉིས་པའི་ཤིང་ཕོ་རྟ་ལོ་སྟེ་སྤྱི་ལོ1714ལོར་སྐུ་གཤེགས།དགུང་ལོ་བཞི་པར་ཡབ་ཆེན་ལས་བཀའ་བརྒྱད་གསང་རྫོགས་ཀྱི་དབང་ལུང་རྣམས་གསན། དགུང་ལོ་བཅུ་གཅིག་པར་ལྷ་སར་ཕེབས་ཏེ་རྒྱལ་དབང་ལྔ་པ་ཆེན་པོར་དབུ་སྐྲའི་གཙུག་ཕུད་ཕུལ། དགུང་ལོ་བཅུ་གསུམ་པར་ཡོངས་སུ་གྲགས་པའི་སློབ་དཔོན་སུམ་ཅུ་རྩ་ལྔར་བསྟེན་ནས་མདོ་སྔགས་རིག་གནས་དང་བཅས་པར་ལེགས་པར་སྦྱངས་ནས་མཁས་པའི་གྲགས་པ་ཐོབ། དགུང་ལོ་བཅོ་བརྒྱད་པར་གཡའ་མ་ལུང་ནས་རིག་འཛིན་ཐུག་ཐིག་གི་ཆོས་སྐོར་དང་། དགུང་ལོ་ཉེར་གཉིས་པར་རྫུ་འཕྲུལ་གྱི་བཀོད་པས་ཤེལ་བྲག་ནས་འཇམ་དཔལ་གཤིན་རྗེ་དྲེགས་འཇོམས་དང་། དགུང་ལོ་སོ་གཅིག་པར་ཨོ་དཀར་ཕྲགས་ནས་གུར་དྲགས་དང་། རྡོར་སེམས།ཨ་ཏིའི་སྐོར། དགུང་ལོ་སོ་ལྔ་པར་ཤ་འུག་ལྟག་སྒོ་ནས་ཐུགས་རྗེ་ཆེན་པོ་བདེ་གཤེགས་ཀུན་འདུས་ཀྱི་ཆོས་སྐོར་སྤྱན་དྲངས་པ་སོགས་ཤིན་ཏུ་མང་སྐད། དགུང་ལོ་སོ་གཅིག་པ་རབ་བྱུང་བཅུ་གསུམ་པའི་མེ་ཕོ་འབྲུག་ལོ་སྟེ་སྤྱི་ལོ1676ལོར་སྨིན་གྲོལ་གླིང་གསར་འདེབས་མཛད་དེ་རྟེན་དང་རྟེན་པ་བཞེངས། དགུང་ལོ་ང་ལྔ་པར་སྒྲུབ་མཆོད་ཁང་དང་། གསང་སྔགས་ཕོ་བྲང་བཅས་གསར་བཞེངས་མཛད། བཤད་སྒྲུབ་ལས་གསུམ་གྱི་སྒོ་ནས་བསྟན་པ་རྒྱ་ཆེར་སྤེལད་ཏོ། ཁོང་གི་མཚན་གྱི་རྣམ་གྲངས་ནི་ཀུན་དགའ་དངོས་གྲུབ་རིན་ཆེན་དབང་རྒྱལ་དང་། པདྨ་གར་དབང་འགྱུར་མེད་རྡོ་རྗེ། ངག་དབང་པདྨ་བསྟན་འཛིན་བཅས་སོ། ཁོང་གི་སློབ་མ་གྲགས་ཆེ་བ་ནི་རྒྱལ་དབང་ལྔ་པ་ཆེན་པོ། སྡེ་སྲིད་སངས་རྒྱས་རྒྱ་མཚོ།རྡོ་བྲག་རིག་འཛིན་པདྨ་འཕྲིན་ལས། ས་སྐྱ་ཁྲི་ཆེན་ཀུན་དགའ་བཀྲ་ཤིས། རྩེ་གདོང་ཞབས་དྲུང་ཁུ་དབོན་བཅས་ཡིན། ཁོང་གི་གསུང་རྩོམ་ནི་པོད་ཆེན་བཅོ་ལྔ་ཙམ་བཞུགས་སོ།།​​​​
